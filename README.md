
This is a robotic lawn mower refurbishment project, starting from a broken one, reusing the perimeter wire and (hopefully) docking station only

Here below current prototype board, available space will be probably filled with charging/docking utility components. All connectors recovered from original card. This one contains essential components: wire sensors with signal conditioners, accelerometer ( to detect tilt and collisions ), motor driver L293D, mower driver ( IRLZ34 )  and the microcontroller ( Arduino Nano ).
